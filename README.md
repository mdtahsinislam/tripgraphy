# 🌍 TripGraphy - Travel & Journey Sharing Platform
## 📸 Screenshot

![TripGraphy Screenshot](https://raw.githubusercontent.com/mdtahsinislam/tripgraphy/main/Travel.png)

# Live link : https://tripgraphy.vercel.app/

🚀 A modern, responsive travel web application built with **Next.js + TypeScript**.  
Users can explore destinations, share their travel experiences, and manage their own travel blogs.

---

## ✨ Features

### 🔐 Authentication
- Secure login system using NextAuth
- Google / Credentials login support
- Protected routes for logged-in users only

### 🧭 Travel System
- 🌍 List View & Map View of destinations
- 🗺️ Interactive destination map
- Travel exploration UI

### 📝 Travel Diaries 
- View all travel posts
- Add your own journey (only logged-in users)
- Manage your personal posts

### 🛍️ Shop Section
- Browse travel-related products
- Manage personal purchases

### 📱 Responsive Design
- Fully responsive (Mobile + Desktop)
- Beautiful dropdown navbar
- Clean UI with Tailwind CSS

---

## 🛠️ Tech Stack

- ⚡ Next.js 
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🔐 NextAuth.js
- 🌐 React Hooks
- 🗺️ Leaflet (Map Integration)
